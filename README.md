# FlexGuard AI 🩺

## Wearable IoT System for Hypermobility Detection

A smart wearable system built using **ESP32, MPU6050, GSR sensor, and Blynk IoT** to monitor joint movement and physiological stress.

---

## 🔧 Hardware Stack
- ESP32
- MPU6050 (Accelerometer + Gyroscope)
- GSR Sensor
- Vibration Motor
- Wi-Fi Module
- Blynk IoT Platform

---

## ⚙️ Key Features
- Real-time motion tracking
- Roll & pitch angle computation
- Stress level monitoring (GSR)
- Haptic vibration alert system
- Cloud monitoring via Blynk


---

## 🚀 Future Scope
- AI-based movement prediction
- Smartwatch integration
- BLE communication
- Health analytics dashboard
